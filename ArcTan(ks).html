ArcTan(ks)
Hackathon's Most Creative Award Winner!
ArcTan(ks) is a game I conceptualized and then developed with our three-person team to win amongst 21 teams and 79 participants in Temple University's OwlHacks Hackathon
Levels
Developed and designed a fully navigational level menu for seamless movement to different menus
Creation of Game and User Interface
Created the game's map interface as well as desinging and placing the level number hints, tips, and menu on the left side
Development of Obstacles
One difficulty we had was creating barriers, so I treated the four sides of obstacles as walls, which can't be touched, making it so that players had to use more creative solutions to win the level
Level Design
Purposefully designed each level to slighlty progress in difficulty while teaching the player one new thing about graphical functions
Implementing Advanced Functions
Implemented full freedom on user input, allowing the user to input any function they wish and output a graphical representation using SymPy
Leaderboard Creation
Used MySQL to set up a basic local leaderboard to increase gamification and competition in order to make a more enjoyable and rewarding experience!
